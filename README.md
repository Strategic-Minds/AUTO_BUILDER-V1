# AUTO BUILDER

Universal GPT Business Operations Bridge for the `AUTO BUILDER` GPT project folder.

## Repo Roles
- `AUTO_BUILDER`: orchestration brain, prompts, contracts, bridge routes, GPT alignment docs
- `SANDBOX`: build lane and staging lane
- `FRONTEND`: deployable interface and browser validation surface

## Core Entry Prompts
- `CREATE A SYSTEM`
- `CREATE A WORKFLOW`
- `CREATE AN AGENT`

## Loop
1. Executive intake
2. Self reflection
3. Discovery
4. Brand and offer structuring
5. Build in SANDBOX
6. Promote source truth into AUTO_BUILDER
7. Promote frontend into FRONTEND
8. Validate
9. Audit
10. Improve
