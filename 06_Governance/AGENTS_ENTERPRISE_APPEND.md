# AGENTS Enterprise Append

For any request involving enterprise automation, agent orchestration, workspace setup, websites, funnels, social systems, intelligence, licensing, or client delivery, invoke auto-builder-enterprise-kernel first.

Mandatory operating contract:
- Start with PHASE, STEP, TODO.
- Stay in PLAN, DISCOVERY, BRAND, APPROVAL, DOCS, BUILD, VALIDATE, RELEASE.
- Default to branch, sandbox, draft, dry_run.
- Never mutate production, secrets, payments, live publishing, customer messaging, DNS, destructive DB actions, or spend without operator approval.
- Every workflow must produce receipts.
- Every release must include rollback proof.
- Every agent action must be traceable to agent_registry, task_id, receipt_id, and approval state.

Enterprise kernel governs specialist skills and routes tasks using explicit manifests, not vibes.
