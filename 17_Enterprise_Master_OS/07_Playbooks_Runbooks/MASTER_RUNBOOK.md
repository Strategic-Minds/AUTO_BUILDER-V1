# Master Runbook

## New business system workflow
1. EXECUTE PLAN MODE
2. EXECUTE DISCOVERY
3. EXECUTE BRAND OPTIONS
4. Operator selects one option
5. EXECUTE DOCS
6. EXECUTE BRANCH SCAFFOLD
7. EXECUTE VALIDATE
8. EXECUTE AUTO HEAL PREVIEW
9. EXECUTE RELEASE REVIEW
10. Operator approves or sends back to repair

## Website workflow
- intake -> source truth -> competitor benchmark -> brand -> content -> template -> preview -> Playwright -> score -> auto-heal -> release review

## Auto Social workflow
- intake -> brand rules -> platform plan -> content calendar -> draft queue -> approval queue -> schedule -> gated publish -> metrics -> optimization

## Intelligence workflow
- source list -> public discovery -> ingest -> classify -> summarize -> chunk -> embed -> Drive mirror -> Supabase source ID -> benchmark -> insight report

## License workflow
- intake -> tenant plan -> entitlement map -> workspace scaffold -> repo/template clone -> onboarding -> support -> renewal
