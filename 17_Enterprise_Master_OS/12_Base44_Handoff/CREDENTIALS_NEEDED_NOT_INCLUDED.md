# Credentials Needed, Not Included

This package intentionally does not contain credentials.

## Needed in provider dashboards
- GitHub token with repo permissions
- Vercel token/project link
- Supabase URL and anon key
- Supabase service role key
- AI Gateway API key
- OpenAI API key if not using AI Gateway only
- Google Workspace OAuth client credentials
- Google service account or delegated Workspace access where approved
- Metricool/Klaviyo/HubSpot/Shopify/Stripe keys only when those systems are approved

## Rule
Never paste secrets into chat logs, docs, GitHub commits, Drive docs, or downloadable ZIP files.
