# Operator Execute Phrases

Use these simple commands to operate the system.

## Safe default commands
- EXECUTE PLAN MODE: create TODO, phase, risks, and discovery path.
- EXECUTE DISCOVERY: inspect source truth and produce verified/inferred/could-not-verify.
- EXECUTE BRAND OPTIONS: produce exactly 3 brand packs, 3 website/content directions, 3 workflow options.
- EXECUTE DOCS: create builder docs after approval.
- EXECUTE BRANCH SCAFFOLD: create branch-safe scaffold only.
- EXECUTE VALIDATE: run QA, Playwright, lint/build, route, security, and receipt checks.
- EXECUTE AUTO HEAL PREVIEW: repair preview branch until score threshold or max iterations.
- EXECUTE RELEASE REVIEW: produce release recommendation, not live deployment.
- EXECUTE OPERATE DRY RUN: run scheduled workflow in dry-run mode.

## Gated commands
These require explicit confirmation:
- EXECUTE PRODUCTION DEPLOY
- EXECUTE SUPABASE MIGRATION APPLY
- EXECUTE GOOGLE WORKSPACE PROVISION
- EXECUTE LIVE SOCIAL PUBLISH
- EXECUTE CUSTOMER MESSAGE SEND
- EXECUTE PAYMENT ACTION
- EXECUTE DNS CHANGE

## Emergency commands
- FREEZE SYSTEM: stop all writes and scheduled actions.
- ROLLBACK PREVIEW: restore last known good branch or preview.
- QUARANTINE OUTPUT: mark artifact unsafe or low confidence.
- AUDIT RECEIPTS: produce full chain of evidence.
