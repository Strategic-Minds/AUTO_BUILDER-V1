# AUTO_BUILDER MASTER OS PACKAGE

Universal branch-safe package for installing the AUTO_BUILDER operating system into a new or existing business stack.

## Mission
Create a governed, agent-operated business factory that can plan, discover, brand, document, scaffold, validate, repair, and prepare release packages for websites, apps, lead-gen systems, Auto Social systems, business operating systems, and licensing offers across any industry.

## Non-negotiable safety model
This package is a scaffold and prompt/control system. It does not include secrets, live credentials, API keys, payment keys, DNS tokens, private account tokens, or production execution authority.

Default mode: branch, sandbox, draft, dry-run, preview.

Production deploys, Supabase migrations, credential changes, customer messages, social publishing, payments, DNS, and spend require operator approval.

## Core runtime
- GPT Business Workspace agents
- Base44 Apex as super-agent/orchestrator
- Strategic-Minds/AUTO_BUILDER repo as governance brain
- GitHub as source control and receipt surface
- Vercel as app/cron/workflow host
- Supabase as memory, queue, receipts, RAG, and state
- Google Shared Drive as human-readable source truth
- Playwright Chromium as headless QA agent
- Vercel AI Gateway as model router
