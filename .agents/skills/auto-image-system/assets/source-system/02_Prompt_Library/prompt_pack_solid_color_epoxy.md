# Prompt Pack: Solid Color Epoxy Floors

## Warehouse Solid Gray

Create an ultra-realistic solid gray epoxy floor in a clean warehouse or light industrial facility. Include uniform gray coating, slight satin sheen, control joints, overhead lights, pallet racks, and realistic wear-safe surface texture. No mirror-like reflection, no fake text, no logos.

## Mechanical Room

Create a realistic solid color epoxy floor in a mechanical room with clean gray coating, equipment pads, floor drain, wall edges, and industrial lighting. The floor should look practical, durable, and professionally installed.

## Commercial Backroom

Create a photorealistic solid tan epoxy or urethane floor in a restaurant backroom or storage area. Include baseboards, shelving, realistic traffic path, and even coating coverage.
