# Prompt Pack: Quartz Epoxy Floors

## Commercial Kitchen Quartz

Create an ultra-realistic quartz broadcast epoxy or urethane floor in a commercial kitchen. Use fine gray and white quartz texture, slip-resistant satin finish, cove base, floor drain, stainless equipment, and overhead fluorescent or LED lighting. The floor should look durable, hygienic, and contractor-installed. No fake logos, no puddles, no CGI texture.

## Clinic / Restroom Quartz

Create a realistic quartz flooring system in a clinic restroom or healthcare corridor. Use light neutral quartz broadcast, seamless cove base, satin finish, clean walls, and realistic commercial lighting. Show a durable non-slip texture without looking rough or dirty.

## Locker Room Quartz

Create a photorealistic quartz epoxy floor in a locker room with drains, benches, cove base, and fine aggregate broadcast. Use a tan, gray, and white quartz blend. The image should feel real, clean, and commercial-grade.
