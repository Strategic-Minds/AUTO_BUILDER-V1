# Prompt Pack: Concrete Countertops

## Modern Kitchen Countertop

Create an ultra-realistic concrete countertop in a modern kitchen. Use natural gray cast concrete, realistic thickness, eased edge profile, sink cutout, subtle pinholes, soft mottling, cabinet shadows, and natural window light. The countertop must look structurally supported and contractor-built. No impossible cantilever, fake marble, warped cabinets, or plastic texture.

## Waterfall Island

Create a photorealistic concrete waterfall island countertop in a high-end kitchen. Include realistic slab thickness, seams, soft gray tonal variation, cabinet alignment, pendant lighting, and natural shadows. Avoid impossible overhangs or distorted geometry.

## Commercial Bar Top

Create an ultra-realistic cast concrete bar countertop in a restaurant or brewery. Use dark gray sealed concrete, subtle aggregate, realistic edge, under-bar shadows, warm lighting, and practical bar setting. No fake logos, fake text, or impossible reflections.

## Bathroom Vanity

Create a realistic concrete vanity countertop with integrated sink, matte sealer, soft mottling, realistic drain, wall tile, faucet shadows, and clean modern bathroom styling.
