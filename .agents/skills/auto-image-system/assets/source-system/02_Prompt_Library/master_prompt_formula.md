# Master Prompt Formula

Use this formula for every image.

## Formula

Create an ultra-realistic contractor-grade image of:

Finish:
`[full flake epoxy / metallic epoxy / quartz floor / solid color epoxy / stained concrete / polished concrete / overlay / microtopping / concrete countertop]`

Space:
`[garage / basement / showroom / restaurant / retail / salon / commercial kitchen / warehouse / patio / pool deck / kitchen / bathroom / office / lobby]`

Surface details:
`[flake size and colors / metallic movement / aggregate exposure / stain tone / overlay texture / countertop edge and cutouts]`

Installation realism:
`[control joints, saw cuts, baseboards, cove base, stem walls, drains, thresholds, cabinets, sink cutouts]`

Lighting:
`[natural daylight, overhead LEDs, showroom lighting, garage door light, warm restaurant lighting]`

Camera:
`[24mm wide interior photo, 35mm natural perspective, 50mm detail shot]`

Quality:
`ultra-realistic, photorealistic, professional contractor portfolio photo, natural shadows, real material behavior`

Restrictions:
`no CGI look, no impossible reflections, no warped room, no fake text, no logos, no plastic texture, no wet floor unless specified, no fantasy pattern`

## Universal Negative Prompt

Avoid: fake CGI, cartoon, illustration, impossible reflections, mirror-like water floor, warped room, distorted cabinets, floating furniture, fake text, logos, unrealistic flake scale, marble veins on flake floor, missing edges, broken baseboards, impossible countertop thickness, plastic-looking concrete, over-saturated colors, messy construction debris unless requested.
