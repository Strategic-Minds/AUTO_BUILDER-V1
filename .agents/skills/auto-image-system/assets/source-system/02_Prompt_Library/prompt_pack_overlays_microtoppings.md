# Prompt Pack: Overlays And Microtoppings

## Modern Microtopping Floor

Create an ultra-realistic decorative concrete microtopping floor in a modern residential interior. Use thin troweled cementitious texture, soft gray tonal variation, subtle hand movement, matte to satin sealer, clean baseboards, and natural daylight. Avoid fake marble, tile seams, CGI smoothness, and impossible shine.

## Restaurant Overlay

Create a photorealistic decorative concrete overlay in a restaurant dining room. Include subtle trowel texture, warm gray finish, sawcut accents, satin sealer, warm lighting, furniture, and realistic floor edges.

## Patio Overlay

Create an ultra-realistic decorative concrete overlay on an outdoor patio with stone-like texture, natural gray and tan color variation, expansion joints, patio furniture, sunlight, and realistic exterior shadows. No fake repeating pattern, no plastic texture.

## Pool Deck Overlay

Create a realistic textured decorative concrete overlay around a pool deck. Use slip-resistant surface, light cool gray color, sawcut joints, coping edge, water nearby, outdoor sun, and realistic non-glare finish. No wet mirror look, no unsafe glossy surface.
