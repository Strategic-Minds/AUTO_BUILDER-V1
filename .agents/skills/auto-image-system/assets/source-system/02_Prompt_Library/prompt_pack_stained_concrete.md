# Prompt Pack: Stained Concrete Floors

## Warm Brown Restaurant

Create an ultra-realistic stained concrete floor in a modern restaurant. Use warm brown and amber translucent stain with natural mottling, subtle slab variation, sawcut joints, and a satin sealer. Include tables, chairs, warm lighting, and realistic concrete character. No fake tile texture, no CGI swirls, no impossible gloss.

## Residential Basement Stain

Create a photorealistic residential basement with stained concrete floor in charcoal, walnut, and soft gray tones. Show natural concrete mottling, expansion joints, satin sealer, baseboards, and comfortable finished-room lighting.

## Retail Stained Concrete

Create a realistic stained concrete retail floor with subtle earth-tone color variation, polished/sealed sheen, sawcut pattern, shelves, natural customer-path perspective, and contractor portfolio quality.
