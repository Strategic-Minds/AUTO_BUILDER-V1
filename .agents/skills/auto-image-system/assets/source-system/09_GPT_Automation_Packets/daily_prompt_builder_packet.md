# Daily Prompt Builder Packet

Run the daily concrete image prompt workflow.

Use:
- `04_Manifests/content_calendar.csv`
- `04_Manifests/image_manifest.csv`
- `04_Manifests/source_image_manifest.csv`
- `02_Prompt_Library`
- `10_Finish_Reference_Cards`

Steps:
1. Identify due content.
2. Select the correct finish prompt pack.
3. Add platform modifier.
4. Create 3 ultra-realistic prompt variants per asset.
5. Add universal negative prompt.
6. Assign filenames.
7. Return manifest rows to add.
8. Mark status `ready_for_generation`.

Do not claim concept images are completed jobs.
