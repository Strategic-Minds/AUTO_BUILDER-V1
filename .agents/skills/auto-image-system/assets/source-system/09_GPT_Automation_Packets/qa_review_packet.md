# QA Review Packet

Review today's generated concrete images.

Use:
- `07_QA_Review/qa_checklist.md`
- `07_QA_Review/rejection_reasons.csv`
- `04_Manifests/image_manifest.csv`

For each image:
1. Identify finish category.
2. Check material accuracy.
3. Check installation realism.
4. Check marketing readiness.
5. Mark `approved`, `revise`, or `rejected`.
6. Add QA notes.
7. If revision is needed, write the corrected regeneration prompt.

Do not mark as published without human approval.
