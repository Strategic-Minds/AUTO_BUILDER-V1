# Operating Model

## Industry Focus

This system serves decorative concrete and concrete coatings contractors offering:

- Residential garage coatings
- Commercial epoxy systems
- Decorative flake floors
- Metallic epoxy
- Quartz broadcast floors
- Solid color epoxy or urethane systems
- Grind-and-seal concrete
- Natural polished concrete
- Stained concrete
- Decorative overlays
- Microtoppings
- Concrete countertops

## Production Loop

1. Read the content calendar.
2. Choose the finish category.
3. Select or request source images.
4. Use the finish-specific prompt pack.
5. Generate 3-5 image prompt variants.
6. Produce ultra-realistic image drafts if generation is available.
7. QA every draft for finish accuracy and contractor realism.
8. Update the image manifest.
9. Move approved images to campaign-ready folders.

## Best Use Cases

- Website hero images
- Garage floor gallery concepts
- Commercial flooring visualizations
- Social media posts
- Before/after concepts
- Sales proposal visuals
- Finish comparison graphics
- Countertop inspiration images
- Seasonal promotions

## Approval Gates

Human approval is required before:

- Publishing images as completed project photos
- Using a customer home, business, or logo
- Claiming a real installation location
- Claiming a finish is a specific manufacturer product
- Using source images from customers or jobsites

## Disclosure Guidance

Use clear labels when needed:

- AI-generated project concept
- Design visualization
- Finish inspiration image
- Concept rendering
- Not an installed project photo

Do not misrepresent generated images as completed customer projects.
