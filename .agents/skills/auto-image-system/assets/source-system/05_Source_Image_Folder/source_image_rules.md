# Source Image Rules

## Approved Inputs

- Your own completed project photos
- Customer-approved project photos
- Manufacturer-approved sample boards
- Licensed stock interior/exterior images
- Approved texture references
- Approved product/color chip samples

## Do Not Use

- Competitor project photos as direct source images
- Customer homes or businesses without permission
- Manufacturer photos without permission
- Logos, vehicles, license plates, addresses, or customer-identifying details
- Images you cannot legally use

## Required Manifest Entry

Every source file must be logged in:

`04_Manifests/source_image_manifest.csv`

Required:
- source image ID
- file name
- rights status
- allowed use
- project type
- finish type
- approval status

## Best Practice

Use real source images for layout, room shape, and finish accuracy. Use GPT to create concept images, not fake completed-project claims.
