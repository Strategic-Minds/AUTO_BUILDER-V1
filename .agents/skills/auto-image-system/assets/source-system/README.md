# Concrete GPT Image Automation System

This is a GPT image production system built specifically for:

- Epoxy full flake floors
- Metallic epoxy floors
- Quartz epoxy / urethane floors
- Solid color epoxy floors
- Stained concrete floors
- Natural polished concrete floors
- Decorative concrete overlays
- Microtoppings
- Concrete countertops

The system is designed to create ultra-realistic contractor-grade marketing images, concept visuals, social posts, website assets, before/after concepts, showroom visuals, and sales presentation graphics.

## Core Goal

Generate images that look like real installed concrete flooring and surfaces, not fantasy renderings.

Every prompt should control:

- Substrate type
- Finish type
- Space type
- Lighting
- Surface texture
- Gloss level
- Flake/quartz/metallic distribution
- Edges, joints, drains, baseboards, walls, cabinets, and real-room details
- Camera angle and lens realism
- Contractor-safe installation accuracy

## Folder Map

1. `01_System_Docs` - operating model, install guide, realism rules
2. `02_Prompt_Library` - finish-specific prompt packs
3. `03_Scheduler` - GPT Scheduler prompts and cadence
4. `04_Manifests` - CSV/JSON tracking system
5. `05_Source_Image_Folder` - approved project photos, samples, and references
6. `06_Output_Queue` - generated images awaiting QA
7. `07_QA_Review` - quality and realism review system
8. `08_Archive` - approved/rejected/completed assets
9. `09_GPT_Automation_Packets` - copy-ready automation prompts
10. `10_Finish_Reference_Cards` - quick finish rules by system

## Recommended Use

Use this with GPT Scheduler to run:

- Daily prompt production
- Optional image generation
- QA review
- Weekly content planning
- Monthly finish-library refresh

## Non-Negotiable Realism Rule

The image must look like a real contractor could install it. Reject images with impossible reflections, fake marble swirls on flake floors, warped rooms, missing control joints, wrong scale flakes, floating objects, incorrect countertop thickness, impossible lighting, or plastic-looking concrete.
