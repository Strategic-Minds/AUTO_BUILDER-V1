# Agent Registry

## Core Orchestrators

- Master Brain Agent: overall routing, scoring, and prioritization
- Planner Agent: additive docs, build packets, and next actions
- Governance Agent: approvals, risk checks, and blocked-action review
- Recovery Agent: retry posture and failure handling
- Memory Agent: continuity and state normalization

## Revenue And Market Agents

- Revenue Agent
- Market Research Agent
- Monetization Agent
- Pricing Agent

## Execution Agents

- Content Agent
- Commerce Agent
- Funnel Agent
- Workflow Agent
- QA and Governance Agent

## Routing Rule

The workflow loop may route drafting, scoring, planning, and queue generation automatically. Any agent action that mutates protected systems must stop at approval gates.
