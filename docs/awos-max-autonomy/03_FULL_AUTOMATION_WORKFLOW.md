DISCOVER -> VALIDATE -> BRAND -> BUILD -> DEPLOY -> DISTRIBUTE -> MONETIZE -> ANALYZE -> OPTIMIZE -> SCALE -> REPLICATE -> REPEAT

## Runtime Translation

1. Rehydrate doctrine and active state.
2. Read current blocker and runtime evidence.
3. Reconcile against the AWOS handoff pack.
4. Materialize the next queue set.
5. Split safe work from protected work.
6. Emit queue and bridge telemetry.
7. Return the next instruction for the recursive-control loop.
8. Repeat every five minutes.

## Protected Execution Stops

The workflow must pause for approval on:
- schema activation
- deployment boundary changes
- billing mutations
- live publishing
- mass outreach
- destructive changes
