# Approval Form Schema

## Fields
- Project name
- Consultation pack version
- Selected brand pack: A/B/C
- Selected website pack: A/B/C
- Selected workflow option: A/B/C
- Approval choice:
  - Approve as-is
  - Approve with exact changes
  - Request one free revision
  - Reject and restart discovery
- Exact changes by section
- Authorization checkbox
- Signature/name
- Email
- Date/time

## Rule
If not approved as-is, the agent must ask for exact field-by-field changes before continuing.
