# Top 3 Benchmark and Reverse Engineering Runbook

## Goal
Find the top 3 relevant public sites/systems for the user's request, benchmark them, and create an original strategy designed to beat them.

## Selection criteria
- Relevance to industry and offer
- Search visibility or reputation
- Ratings/reviews if public
- Conversion quality
- Design quality
- Content depth
- Automation maturity
- Trust proof

## Public data allowed
- route map
- page structure
- CTA patterns
- public pricing/offers
- SEO metadata
- public schema
- public form fields
- performance signals
- mobile/PWA behavior
- trust proof categories

## Blocked
- copying code/assets/logos/images/reviews/body copy
- login-only or paywalled data
- bypassing robots, CAPTCHA, rate limits, or access restrictions
- private analytics or private customer data

## Output
- competitor selection memo
- benchmark matrix
- opportunity gaps
- original reconstruction plan
- implementation recommendations
- metrics to beat
