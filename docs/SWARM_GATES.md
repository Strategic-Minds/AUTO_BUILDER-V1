# Swarm Gates

## Allowed without approval
- Research
- Classification
- Drafting docs
- Creating branch-safe scaffold plans
- Dry-run MCP calls
- Writing receipts in sandbox/preview
- Creating queue items

## Requires approval
- Creating live Drive folders
- Provisioning accounts or emails
- Applying Supabase migrations
- Deploying production
- Sending customer messages
- Publishing social posts
- Charging money
- Buying tools/domains
- Deleting data
- Changing secrets

## Required receipt fields
- run_id
- phase
- skill
- agent_id
- action
- source
- destination
- dry_run
- approval_id
- result
- rollback_plan
- created_at
