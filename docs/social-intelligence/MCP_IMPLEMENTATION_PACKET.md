# 5-Minute Workflow Contract

Every 5 minutes, read-only or receipt-only unless separately approved:

1. Collect signals.
2. Normalize sources.
3. Score trends.
4. Score audiences.
5. Score keywords/phrases.
6. Score creator/competitor movements.
7. Score image/creative patterns when assets exist.
8. Rank opportunities.
9. Route opportunities to personas and faceless pages.
10. Generate draft recommendations.
11. Store memory.
12. Generate receipt.
13. Alert operator only when configured and approved.

No public posting, comments, replies, messages, paid actions, or external writes occur in this cron without approval.

# Scoring Formulas

## Opportunity Score
Opportunity Score =
(Trend Velocity * 0.20) +
(Audience Match * 0.18) +
(Shareability * 0.14) +
(Authority Potential * 0.12) +
(Conversion Potential * 0.12) +
(Platform Fit * 0.10) +
(Creative Fit * 0.08) +
(Evidence Confidence * 0.06) -
(Competition Saturation * 0.10) -
(Risk Score * 0.10)

## Trend Score
Trend Score =
Velocity + CrossPlatformSpread + CreatorAdoption + AudiencePull + Novelty - Saturation - Decay

## Audience Match Score
Audience Match =
PainPointFit + DesireFit + IdentityFit + BelongingFit + PersonaFit + BuyerIntent

## Content Score
Content Score =
HookStrength + RetentionPotential + Clarity + Shareability + CTAFit + PlatformFit + BrandFit - Risk

## Report Confidence Score
Report Confidence =
SourceQuality + SourceCount + Freshness + CrossSourceAgreement + EvidenceTraceability - SpeculationPenalty

# Blocker Elimination Plan

## Common Failure: Missing Platform API
Workaround: use manual export, public evidence capture, browser screenshot receipt, or platform-approved integration.

## Common Failure: Missing Analytics
Workaround: import CSV/export, creator studio screenshots, or Google Sheet manual ledger.

## Common Failure: Social Posting Blocked
Workaround: create approval queue and draft-only content packet.

## Common Failure: Gmail/Drive/Docs/Sheets Connector Blocked
Workaround: generate local docs, CSVs, or repo handoff packets.

## Common Failure: Trend Data Incomplete
Workaround: cross-check Google Trends, Reddit, YouTube, X, news, creator feeds, and manual source receipts.

## Common Failure: Report Hallucination Risk
Workaround: every claim must have evidence status: Verified, Inferred, or Could Not Verify.
