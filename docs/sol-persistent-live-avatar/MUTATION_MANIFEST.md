# SOL Mutation Manifest

## Authorized Scope
- GitHub
- Vercel
- Supabase staging
- Shopify
- Google Workspace
- OpenAI
- HeyGen
- Xyla
- Opus
- Social scheduling systems
- Crons, workflows, and sandbox automation

## Restricted Until Additional Approval
- Production database destruction
- Billing escalation
- Irreversible payment operations
- Mass outbound social posting
- Production environment deletion

## Required Logging
Every mutation must record:
- Timestamp
- System touched
- Actor
- Approval source
- Rollback path
- Result
- Blockers and workarounds
