# Approval and Gate Checklist

## Always allowed
- Discovery
- Source truth reading
- Draft docs
- Branch-only changes
- Dry runs
- Smoke tests
- Preview validation

## Approval required
- Production deploys
- Secrets
- Payments or spend
- Customer messages
- Live social publishing
- Database writes outside reviewed migrations
- Destructive actions
- Public Drive sharing

## Receipt required
- Discovery receipt
- MCP dry_run receipt
- Build receipt
- Smoke receipt
- Approval receipt
- Rollback receipt
- Release receipt
