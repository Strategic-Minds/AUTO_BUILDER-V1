# AUTO BUILDER OS - Drive And Git Source Truth Policy

Every new idea, system change, workflow change, agent change, frontend change, backend change, or Auto Social change must search both Git and Drive before execution.

## Git Required

Search canonical repo docs, current branch, open PRs, and evidence files.

## Drive Required

Search `AUTO BUILDER DOCS`, `AUTO_BUILDER`, project folders, workbook sources, and approved brand/source documents.

## If Drive Write Is Unavailable

Record a hard gate and create Drive-ready content in Git until the Drive write bridge is available.
