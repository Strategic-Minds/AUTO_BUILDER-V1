# AUTO BUILDER OS - Build Evidence Requirements

Every completed checklist item must have evidence.

## Required Evidence Types

- file path
- code path
- PR link
- commit SHA
- smoke result
- screenshot
- Supabase receipt
- connector dry-run receipt
- hard-gate receipt
- approval receipt
- rollback note

## Invalid Evidence

- "It should work"
- env-name presence alone
- static mock output
- sample data
- unverified v0 claim
- unrun test command
- undocumented manual action
