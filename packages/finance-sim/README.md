# Finance Simulation Package

This package holds the financial prediction and simulation scaffold for AUTO BUILDER.

## Current contents
- `simulator.py` for deterministic and Monte Carlo-style scenario simulation

## Intended use
- run reproducible scenario simulations with fixed seed control
- compare downside, base, and upside cases
- generate NPV, loss probability, runway, and decision signals
- feed executive approval and calibration workflows

## Governance rule
Simulation output is decision support only. No autonomous spend, investment, or financial commitment is allowed.
